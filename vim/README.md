# Vim

Single-file, zero external plugins, minimal Vim config.

This is intended to be used on remote machines, especially if they have strict
security and don't allow third party plugins.

To download and install this file without needing other dotfiles:

```bash
curl -fsSLo "$HOME/.vimrc" https://raw.githubusercontent.com/peter-bread/.dotfiles/refs/heads/main/vim/vimrc
```

```bash
wget -qO "$HOME/.vimrc" https://raw.githubusercontent.com/peter-bread/.dotfiles/refs/heads/main/vim/vimrc
```
